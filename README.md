## Features

- **Budget Management:** Set your budget and monitor your spending against it.
- **Expense Tracking:** Add, edit, and categorize expenses.
- **Savings Overview:** Instantly see how your savings compare to your budget.
- **Search and Filter:** Easily find specific expenses using search and filter options.

## Technologies Used

- React: Building the user interface and managing application state.
- Bootstrap: Styling and responsive design.
- Local Storage: To persist budget and expenses data.
